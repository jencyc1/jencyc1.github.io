I calculated the the resistor necessary for the red LEDs using the V=IR equation. R = V/I = (5V-1.8V)/0.02A = 160 ohms. Thus, I used the next closest resistor which is 220 ohms. I used a 10k resistor for the photoresistor because with the voltage resistor equation, R2 = 10,000, but R1 varies based on the brightness. Using the voltage divider formula, we can see that the highest brightness value (around 800) does not go above the voltage or drain the current. The same logic applies for the thermistor.
